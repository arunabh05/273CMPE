Audio tag in HTML5 Tutorial

The audio tag in HTML5 can be used to embed the audio files in a web page without using any additional plug-ins. The audio tag requires a source tag which describes the source of the audio file and also the format of the file.

Syntax : <audio controls> <source src="audio.mp3" type="audio/mpeg"> </audio>

Video tag in HTML5 Tutorial

The video tag in HTML5 can be used to embed the a video in a web page. The video tag requires a source tag which describes the source of the file and also the format of the file. The height and width of the video player can also be defined using the height and width attributes in video tag

Syntax : 
	 <video width="320" height="240" autoplay>
		<source src="movie.mp4" type="video/mp4">
  	        <source src="movie.ogg" type="video/ogg">
	 </video>
